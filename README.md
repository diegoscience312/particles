# Sistema de Partículas 3D con Interacción por Mano ✋✨

Controla un sistema de partículas 3D moviendo tu mano frente a la webcam.
Hecho con **Three.js** (render 3D) y **MediaPipe Hands** (detección de mano).

## 🎮 Cómo se usa

1. Abre la página y permite el acceso a la **cámara**.
2. Pon tu mano frente a la webcam (se ve en la esquina inferior derecha).
3. Controles:
   - **Mover la mano** → rota la figura.
   - **Cerrar el puño** → acerca o aleja el puño de la cámara para cambiar el tamaño (zoom).
   - **Abrir la mano** → vuelve al tamaño normal.
4. Panel de la izquierda: cambia la **forma** (Árbol, Saturno, Corazón) y el **color**.
5. También puedes rotar con el **ratón** (arrastrar) y hacer zoom con la **rueda**.

> ℹ️ La cámara solo funciona sobre **HTTPS** o **localhost**. Si lo abres con el link de GitHub Pages funciona perfecto.
